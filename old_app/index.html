TuxBlocks

For one of my classes this semester (Spring 2014), I took Game Based Learning and was invited to help out with a tower defense game called TuxBlocks which aims to make practicing algebra fun. Thomas and I have been working on extending the game to include a story mode which would teach and mentor algebra, not just be a supervised practice. That repo is on GitHub and was originally created during the 2013 Google Summer of Code. I'm really excited with making the game smart, that is, having a student model that can customize the experience to the student, optimizing engagement and learning gains.

FindBugs

One of my side projects is a plugin for FindBugs, a Java static analysis tool. I have forked fb-contrib to experiment with some new detectors and to play with the messages. My co-workers' research has shown that one reason static analysis tools aren't used is because of their unapproachable language. My fork aims to add better messages, that include details of how to fix the problem.
